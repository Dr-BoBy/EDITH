# EDITH
Project M1 SETSIS

Launch : 
  - python Main.py

Command :
  - l : enter live mode 
  - f : enter face detect mode (default)
  - o : enter object detect mode (Supported : Background, Plane, Bike, Bird, Boat, Bottle, Bus, Car, Cat, Chair, Cow, Table, Dog, Horse, Motorbike, Person, Plant, Sheep, Sofa, Train, Monitor)
  - q ; quit
